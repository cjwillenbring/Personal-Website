WEB PROJECTS

THIS WEBSITE
+

I made this website entirely from scratch with a little help from google and Stack Overflow when niche problems arose. In hindsight, using outisde tools such as bootstrap would've been a big help, but it was a pleasant challenge to take on this project with the limited toolset as it forced me to learn my tools better and further develop my web stack. I am always updating this webpage, but I have finished much of the heavylifting already. A lot of what I do now consists of small tweaks and minor UI/Quality of Life improvements.

WIDGETS
+

Learning all the caveats of javascript, jQuery, CSS, and HTML seems almost impossible to manage at first, so I started out building simple webpages and widgets using different syntax and skills each time until I started to build habits in my coding. Despite my habits, I still enjoy working on these mini-projects as they provide a great ROI when it comes to satisfaction in respect to the time put into working on them.

FIRST ITERATION
+

I thought it would be nice to include the first iteration of my website to get a grasp on the site before and after I made the quite sizable renovation. Originally, the site was simply a harsh purple background with some text and a playable tic-tac-toe board that didn't even include the javascript logic to check for a winner. Nevertheless, it was still an interesting task given as homework for an intro level computing class.
